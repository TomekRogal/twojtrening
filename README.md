# Your training

An application to plan your training, with UIs for users, training plans and admins.


The aim of the project is to create an application in which everyone can plan their own set of gym workouts.

# Technologies
- Spring Boot
- Spring Data JPA
- Hibernate
- Spring Form
- Views: 
  - JSP files
- Tests:
  - JUnit
  - Mockito
  - MockMvc
